Part 1.2: Finite Difference Operator

We implement edge detection using finite difference operators D_x = [1, 0, -1] and D_y = [[1], [0], [-1]]. We convolve the cameraman image with these operators to compute partial derivatives in x and y directions. The gradient magnitude is computed using sqrt(dx² + dy²). Finally, we binarize the gradient magnitude using an appropriate threshold to create an edge image, balancing between capturing real edges and suppressing noise.

The threshold (0.3) was chosen qualitatively to preserve important structural edges (camera outline, tripod legs, building edges) while suppressing most background noise. Some fine texture details are lost, but the major edges that define object boundaries are clearly preserved, making this a reasonable trade-off for edge detection.

Part 1.3: Derivative of Gaussian (DoG) Filter

To reduce noise in edge detection, we first apply Gaussian smoothing before computing derivatives. We can either (1) blur the image then apply derivative filters, or (2) create derivative of Gaussian filters and convolve directly. Both approaches are mathematically equivalent due to the associativity of convolution, but the second is more computationally efficient. We use cv2.getGaussianKernel() to create a 1D Gaussian and take the outer product with its transpose to get a 2D Gaussian kernel.

Part 2.1: Image "Sharpening"

We implement unsharp masking to sharpen images. The Gaussian filter acts as a low-pass filter that retains only low frequencies. By subtracting the blurred version from the original, we extract the high frequencies. Adding these high frequencies back to the original image makes it appear sharper. The formula is: sharpened = original + α × (original - blurred), which can be combined into a single convolution operation called the unsharp mask filter.

Part 2.2: Hybrid Images

We create hybrid images by combining the high frequency components of one image with the low frequency components of another. This creates images that look different when viewed from different distances. Up close, you see the high-frequency image, while from far away, you see the low-frequency image. We use Gaussian filters to separate frequency bands and show both spatial and frequency domain representations.
